FOOS - Fast Online Ordering System

For this Coursera/Meta HTML & CSS portfolio project, I decided to revisit the project I had in college, circa 2008. The project was pre-Iphone apps. I created a shopping cart system that made use of SMS technology and cryptography to enable mobile phones to order food/services when mobile apps and the Iphone App Store did not exist. yet. In creating the above sites, I use the FOOS branding.
